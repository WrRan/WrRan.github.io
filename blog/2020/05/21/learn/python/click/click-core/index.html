Step NINE
总结一下，当执行指令python hello.py --count=3时，我们依次执行了：

  1. _make_command(hello, None, {}, Command)
  2. Command(name=hello.__name__.lower().replace(‘_’, ‘-’), callback=hello, params=?, {‘help’: inspect.getdoc(hello)})
  3. Command.__call__
  4. Command.main(self, args=None, prog_name=None, complete_var=None, standalone_mode=None, **extra)
  5. Command.invoke(ctx)
  6. Context.invoke(self.callback, **ctx.params)
  7. callback(*args, **kwargs) (Context.invoke)
