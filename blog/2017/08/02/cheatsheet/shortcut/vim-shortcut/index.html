VIM常用快捷键

2017-08-02 | cheatsheet ， shortcut | |
1,468

Vim - the ubiquitous text editor

光标移动

按键功能
h光标向左移动
j光标向下移动
k光标向上移动
l光标向右移动
Ctrl + f向下移动一页
Ctrl + b向上移动一页
Ctrl + d向下移动半页
Ctrl + u向上移动半页
+光标移动到非空格符的下一列
-光标移动到非空格符的上一列
<n><space><n>表示数字；光标向前移动<n>个字符
0移动到该行的最前面字符处
$移动到该行的最后面字符处
H光标移动到这个屏幕的最上方一行的第一个字符
M光标移动到这个屏幕的中央一行的第一个字符
L光标移动到这个屏幕的最下方一行的第一个字符
G光标移动到该文档的最后一行
<n>G<n>为数字。移动到该文档的第<n>行
gg移动到该文档的第一行
<n><enter><n>为数字。光标向下移动n行

搜索与取代

按键功能
/<word>向光标之下寻找字符串<word>
?<word>向光标之上寻找字符串<word>
n重复前一个搜索的动作
N反向进行搜索动作
:<start>,<end>s/<old>/<new>/g<start>与<end>为数字。在第<start>行至<end>行搜索字符串<old>并替换为字符串<new>
:<start>,$s/<old>/<new>/g与上一个指令相同，不同之处在于$表示最后一行
:<start>,$s/<old/<new>/gc与前两个指令相同，不同之处在于需要用户确认

删除复制粘贴

指令功能
x向后删除一个字符
<n>x<n>为数字。向后删除<n>个字符
X向前删除一个字符
<n>X<n>为数字。向前删除<n>个字符
dd删除游标所在的那一整行
<n>dd<n>为数字。删除游标所在的向下<n>行
d1G删除光标所在到第一行的数据
dG删除光标所在到最后一行的数据
d$删除光标所在到该行最后一个字符的数据
d0删除光标所在到该行第一个字符的数据
yy复制游标所在的那一行
<n>yy<n>为数字。赋值游标所在的向下<n>行
y1G复制光标所在行到第一行的数据
yG复制光标所在行到最后一行的数据
y0复制光标所在到该行行首的数据
y$赋值光标所在到该行行末的数据
p将已复制的数据在光标的下面粘贴
P将已复制的数据在光标的上面粘贴
J将光标所在行与下一行数据合并为一行
c重复删除多个数据，如向下删除10行，则指令为10cj
u撤销前一个动作
Ctrl + r重做上一个动作
.重复前一个动作

模式切换

按键功能
i进入插入模式。从光标所在处插入
I进入插入模式。在当前行的第一个非空格符开始插入
a进入插入模式。从光标所在的下一个字符处开始插入
A进入插入模式。从光标所在行的最后一个字符处开始插入
o进入插入模式。在光标所在的下一行插入新的一行
O进入插入模式。在光标所在的上一行插入新的一行
r进入替换模式。取代光标所在的那一个字符一次
R进入替换模式。一直取代光标所在的文件，直到按下ESC为止

指令模式

指令功能
:w将数据写入硬盘
:w!强制写入数据（需要权限）
:q离开VI
:q!强制离开VI，不保存所做变更
:wq存储并离开VI
ZZ若档案没有变更，则不存储离开；若变更，则存储离开
:w <filename>将数据另存为另一个档案
:r <filename>在编辑的数据中，读入另一个档案的数据，即将<filename>的内容追加到光标所在行的后面面
:<start>,<end> w <filename>将第<start>行至第<end>行的内容存储为文件<filename>
:! <command>暂时离开VI到指令列模式下执行<command>，如:! ls

区块选择

指令功能
v字符选择，会将光标经过的地方反白选择。
V行选择，会将光标经过的行反白选择
Ctrl + v区块选择，可以用长方形的方式选择资料

多文档编辑

指令功能
:n编辑下一个档案
:N编辑上一个档案
:files列出目前这个VIM的开启的所有档案

多窗口功能

指令功能
:sp增加一个窗口，并在该窗口下打开原先的文档
:sp <filename>增加一个窗口，并在该窗口下打开文档<filename>
Ctrl + w + j光标移动到下方的窗口
Ctrl + w + k光标移动到上方的窗口

VIM环境设定与记录

指令功能
:set nu显示行号
:set hlsearch高亮显示搜素结果
:set autoindent自动缩进
:set backup自动备份
:set ruler显示状态栏
:set showmode显示模式

上述指令，可以通过在指令前加no，就可以起到关闭的功能了。如不显示行号，则指令为:set nonu。

指令功能
:set backspace={0,1,2}决定在编辑模式下，退格键可以删除的字符。为2时，可以删除任意值；为0或1时，仅可以删除刚刚输入的字符，而无法删除原本就已经存在的文字
:set all显示目前所有的环境参数设定值
:set显示与系统默认值不同的设定参数
:syntax on依据语法显示不同颜色
:syntax off不显示不同的颜色
:set bg=dark改变显示的颜色色调为深色
:set bg=light改变显示的颜色色调为浅色
