JavaScript is not Script of Java.

概述

话是这么说，但是JavaScript可以通过原型链以及一些技巧实现与Java类似的面向对象机制。下面列出JavaScript权威指南中的例子以供理解。同时，强烈建议大家先阅读JavaScript查缺补漏の女娲补天 - 原型链。

复数

在Java中类成员可以分为如下几种：

名称含义
实例字段基于实例的属性或者变量，用以保存独立对象的状态
实例方法类的所有实例所共享的方法，由每个独立的实例调用
类字段这些属性或变量是属于类的，而不是属于某一个具体实例
类方法这些方法属于类的，而不是属于某个实例

而在JavaScript中函数也是变量，所以上述的四个分类变为了两类，即实例属性和类属性。
在JavaScript中，这些概念的区分需要三个对象的参与，即构造函数对象、原型对象和实例对象。

对象映射关系
构造函数对象任何添加到这个构造函数对象中的属性都是类字段和类方法
原型对象原型对象的属性被类的所有实例所继承，如果这个属性是函数的话，则成为实例方法；如果不是函数，则成为默认实例属性
实例对象定义在实例上的属性，都只是这个实例的属性，所以实例对象上的属性就是实例属性和实例方法

JavaScript中未介绍的可以模拟Java的一些特性：

  1. 对于Java类的实例方法来说，实例字段可以用作局部变量，而不需要使用关键字this来引用他们
    (其实可以使用with来模拟，但是不推荐)
  2. Java中使用final声明字段为常量
    (可以使用闭包中的局部变量来模拟)
  3. Java中的private字段和方法
    (常量属性可以在ECMAScript 5中直接实现)

针对上述说的2、3，在JavaScript中往往通过约定来约束：
常量字段 使用全大写命名
私有字段 使用_开头
